Extension License Manager

Downloads

﻿In diesem Artikel werden die verschiedenen Runtime Packages für die 365 business development Extension License Erweiterung, für Microsoft Dynamics 365 Business Central veröffentlicht.

Ein Runtime Package ist ein vorkompiliertes Anwendungspaket, dass Sie in Ihrer Microsoft Dynamics 365 Business Central On-Premise Installation verwenden können. Hierbei ist es wichtig die korrekte Version, für die bei Ihnen eingesetzte Plattform Version von Microsoft Dynamics 365 Business Central, zu verwenden.

Anwendungsinformationen

Version: 2.0.118.16431
Zuletzt aktualisiert: 06.09.2024

Hinweis
Die aktuelle Dokumentation zu dieser Version finden Sie unter https://docs.365businessdev.com/.

Installation

Verwenden Sie folgenden Befehl zur Installation der 365 business development Extension License Erweiterung für Microsoft Dynamics 365 Business Central, mit dem 365 business development App Installer:

.\Install-Apps.ps1 -appIds @(
	'fcfc9bac-8f9b-402f-9e64-30a8287bc78f'  # 365 business development Extension License by 365 business development
)
